# Athul VS Portfolio

A modern, high-performance personal portfolio built with React, Vite, and Tailwind CSS. This project showcases premium web design aesthetics featuring smooth scrolling, scroll-triggered animations, and a rich, responsive interface.

## Overview

The Athul VS Portfolio is designed to highlight professional experience, skills, and selected projects. It uses a custom four-color palette (Concrete, Craft, Bright Brick, Graphite) tailored for an elegant and premium user experience. The application architecture ensures high performance across mobile and desktop devices with zero layout shifts or stuttering.

## Features

- **Dynamic Sections**: Includes Hero, About, Skills, Experience, Projects, and Contact sections.
- **Premium Aesthetics**: Implements modern web design principles with a bespoke color scheme, modern styling, and high-fidelity typography.
- **Smooth Animations**: Utilizes Framer Motion for rich micro-interactions, scroll-triggered animations, and fluid navigation components.
- **Responsive Design**: Fully optimized for performance and visual consistency across all screen sizes, including extensive mobile optimizations.
- **Interactive Projects**: Clickable project cards with direct integration to external resources.

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Getting Started

### Prerequisites

Ensure you have Node.js installed on your system.

### Installation

1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Install the dependencies:

```bash
npm install
```

### Running the Application

To start the local development server, run:

```bash
npm run dev
```

The application will be available at the local URL provided by Vite.

## Project Structure

- `src/components/`: Contains all modular UI components such as `Navbar.jsx`, `HeroSection.jsx`, `ProjectsSection.jsx`, etc.
- `src/index.css`: Includes global base styles and custom utility styling.
- `tailwind.config.js`: Custom Tailwind CSS configuration including the defined color palette and customized theme settings.

## Scripts

- `npm run dev`: Starts the Vite development server.
- `npm run build`: Builds the application for production.
- `npm run lint`: Runs ESLint to check for code quality issues.
- `npm run preview`: Locally previews the production build.
